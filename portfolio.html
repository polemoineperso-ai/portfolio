01
Générateur de candidature IA
Lettre · Mail d'accompagnement · Profil ATS
✓ En ligne

Un fichier HTML autonome qui génère en 20 secondes une lettre de motivation structurée, deux versions de mail et un profil CV optimisé ATS — à partir du profil candidat et du texte de l'annonce. Quatre passes API en série et en parallèle, audit qualité automatique côté client.

  • Génération en 4 passes : rédaction → correction → mail → analyse ATS
  • Audit automatique : mots interdits, tirets longs, score qualité /10
  • Profil sauvegardé localement — zéro serveur, zéro installation
  • Export PDF intégré, dark mode, trois tons (associatif / privé / public)
HTML vanilla JavaScript Claude API localStorage Netlify

02
Agent de veille emploi automatisé
Python · Windows Scheduler · Gmail OAuth
✓ En production

Un agent Python qui tourne chaque matin à 7h00 sans intervention humaine : scraping multi-sources, scoring IA par profil, email digest HTML actionnable. 180 offres agrégées par jour, 2 livrées en boîte mail avec justification et actions en un clic.

  • 3 sources : France Travail API (8 dép. IDF), Adzuna, Cadremploi RSS
  • Scoring Claude Sonnet : score /10 + tag + justification par offre
  • Email digest HTML — archiver, sauvegarder, générer une candidature
  • Déduplication intelligente, relance automatique, alertes d'erreur
Python 3.13 Claude API Gmail OAuth 2.0 Windows Scheduler BeautifulSoup feedparser

03
Newsletter IA automatisée — NorthIA
Make · Claude · DALL-E 3 · Brevo
✓ En production

Pipeline complet d'automatisation éditoriale : de la capture d'article RSS à la newsletter en boîte mail, sans intervention humaine. Claude enrichit chaque article, génère l'édito et décrit l'image. DALL-E 3 produit un visuel lié au contenu. Brevo distribue.

  • Collecte automatique via RSS + capture URL mobile (Tally)
  • Enrichissement Claude : résumé, mots-clés, angle éditorial, score de priorité
  • Images éditoriales générées par IA, liées au contenu de chaque article
  • 8 catégories avec palette couleur dédiée, édito original à chaque édition
