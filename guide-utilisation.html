Avant de commencer

Vous aurez besoin d'une clé API Anthropic pour utiliser cet outil. Elle est gratuite à créer et vous offre ~5 $ de crédits pour démarrer (environ 50 à 80 générations). Créez votre compte sur console.anthropic.com → "Get API Keys" → "Create Key".

Mode d'emploi

4 étapes pour votre première candidature

L'outil fonctionne entièrement dans votre navigateur. Aucune donnée n'est stockée en ligne — tout reste sur votre ordinateur.

01
Entrez votre clé API
Au premier lancement, un encadré orange vous demande votre clé API.
Copiez-collez votre clé (commence par sk-ant-...) puis cliquez Enregistrer.
Elle est mémorisée dans votre navigateur — vous n'aurez plus à la saisir.
Astuce : sur console.anthropic.com, cliquez "API Keys" dans le menu gauche → "Create Key". Donnez-lui un nom clair (ex: "Candidature IA").
02
Renseignez votre profil Une seule fois
Remplissez vos informations personnelles, puis collez le texte de votre CV et de votre profil LinkedIn.

Pour votre CV PDF : ouvrez-le, faites Ctrl+A puis Ctrl+C, et collez dans le champ.
Pour LinkedIn : allez sur votre profil → "Plus" → "Enregistrer en PDF", puis copiez le texte.

Ajoutez optionnellement un contexte de recherche (secteurs visés, disponibilité, points forts à mettre en avant).
Important : votre profil est sauvegardé automatiquement dans votre navigateur. Vous n'aurez à le remplir qu'une seule fois, même si vous fermez et rouvrez l'outil.
03
Choisissez vos paramètres
Mode : "Répondre à une offre" (collez le texte de l'annonce) ou "Candidature spontanée" (décrivez l'entreprise et le poste visé).

Longueur : Court (200-270 mots), Standard (260-330 mots) ou Long (330-400 mots).

Ton : Adaptez selon le secteur — 🤝 Associatif pour les associations et fédérations, 🏢 Privé pour les entreprises (orienté résultats), 🏛 Public pour les collectivités et administrations.

Règles d'écriture : activez/désactivez les chips (mots interdits, tirets longs, chiffres inventés...) ou ajoutez vos propres instructions dans le champ libre.
04
Générez et exportez
Cliquez "Générer lettre + mail + profil ATS". En 15 à 25 secondes, vous obtenez :

— La lettre de motivation avec audit qualité automatique et score
— Deux versions de mail d'accompagnement (sobre et personnalisée)
— Un profil CV optimisé ATS avec checklist de mots-clés

Utilisez les boutons Copier pour chaque élément, ou Imprimer / PDF pour exporter la lettre mise en page.
Pas satisfait(e) ? Cliquez "↻ Régénérer" pour obtenir une nouvelle version, ou "✏️ Éditer" pour modifier la lettre directement dans l'outil.
Résultats

Ce que l'outil génère

À chaque génération, trois livrables distincts sont produits en parallèle.

📝
Lettre de motivation
Structurée, personnalisée et auditée automatiquement. Score qualité, détection des mots interdits, compteur de mots. Export PDF intégré.
📧
Mail d'accompagnement
Deux versions : sobre (3 lignes efficaces) et personnalisée (ancrée sur l'entreprise). Objet prêt à copier.
🎯
Profil CV optimisé ATS
Titre de poste adapté, profil court et long avec mots-clés de l'annonce intégrés. Checklist de conformité.
Budget

Combien ça coûte ?

L'IA est facturée au token — la consommation réelle est très faible pour ce type d'usage.

Action Coût estimé Avec 5 $ de crédits
Une génération complète (lettre + mail + ATS) ~0,06 à 0,10 € ~50 à 80 générations
Régénération d'une lettre seule ~0,04 € ~120 régénérations
Recharge manuelle 10 € 10 € ~100 à 150 générations supplémentaires

Les crédits sont rechargés manuellement sur console.anthropic.com → Billing. Aucun prélèvement automatique par défaut.
